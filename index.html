What's covered in M06

  • Deuterostome synapomorphies — radial cleavage, enterocoely, blastopore → anus
  • Echinodermata — water vascular system, pentaradial symmetry, CaCO₃ endoskeleton; 5 classes
  • Chordate synapomorphies — notochord, dorsal hollow nerve cord, pharyngeal slits, post-anal tail; 3 subphyla
  • Vertebrate innovations timeline — cranium → jaws → bony endoskeleton → lungs → limbs → amniotic egg → endothermy
  • 7 vertebrate lineages — hagfishes, sharks, ray-finned fish, lobe-fins, amphibians, mammals (3 types), reptiles + birds
  • Convergent evolution — flight (3×), viviparity (100×), endothermy (2×), limb loss
Quick reference

H-W Equilibrium

p + q = 1
p² + 2pq + q² = 1

q = √(freq aa) → p = 1−q → 2pq = heterozygote frequency

Germ Layer Cheatsheet

Ecto = outside → skin, CNS, sense organs

Meso = middle → muscle, bone, kidneys, heart

Endo = inside → gut, liver, lungs, pancreas
